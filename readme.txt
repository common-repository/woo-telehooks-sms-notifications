== Installation ==

1. Install Telehooks SMS Notification either via the WordPress.org plugin repository, or by uploading the files to your server
2. After activating the plugin, Please follow instructions.
3. Thats it!!

== After Installation ==

1. Click on menu link Telehooks SMS.
2. If you don't have Telehooks account,register on http://telehooks.com/register, link (Register Here) is given in admin panel.
3. After registration,Telehooks loginid and Password will be sent to your registered email. 
4. Enter Telehooks loginid, Password , Consumer Key , Secret Key in the form and click on Connect to Telehooks.
